# **App Name**: Gemini Chat

## Core Features:

- User Authentication: Secure user login and signup using Firebase Authentication.
- Interactive Chat Interface: A clean, intuitive interface for sending messages and receiving AI responses, mimicking the style of ChatGPT.
- AI Chatbot powered by Gemini API: Utilize the Gemini API for generating intelligent and context-aware responses.
- Persistent Chat History: Store and retrieve user-specific chat history using Firestore database for continuous conversations.
- User Profile: A section for users to manage their profile information.
- Light/Dark Mode: Allow users to switch between light and dark themes for optimal viewing experience.

## Style Guidelines:

- Primary color: Soft blue (#64B5F6) to create a calm and modern atmosphere. 
- Background color: Light gray (#F0F4F7) for light mode and dark gray (#263238) for dark mode to provide a clean and comfortable backdrop. 
- Accent color: Teal (#4DB6AC) to highlight interactive elements and calls to action.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern, machined, objective, neutral look; suitable for headlines or body text.
- Use simple, outlined icons in a consistent style to represent actions and options.
- Maintain a clean, minimalist layout with ample whitespace to improve readability and reduce clutter.
- Use subtle transitions and animations for a smooth and engaging user experience.
